# StudyReceipt 

A command-line study efficiency analyzer that turns your study session into a receipt.

## What it does

Enter your subjects, hours studied, and marks scored. StudyReceipt calculates an **Efficiency score** (marks per hour) for each subject using Pandas, prints a humorous verdict based on your average efficiency, and shows a Matplotlib bar chart comparing subjects.

## Tech Stack

Python, Pandas, Matplotlib

## Run it

pip install pandas matplotlib
python study_receipt.py
